# 数据库设计

来源：[docs/系统设计方案.md](../系统设计方案.md)

AI 读取范围：核心业务表和辅助表设计，包括用户、车辆、接入事件、媒体、案件、AI 结果、违章和通知等。

---

## 6. 数据库设计

### 6.1 核心表

#### `users`

| 字段 | 说明 |
| --- | --- |
| `id` | 用户 ID |
| `username` | 登录名 |
| `password_hash` | 密码哈希 |
| `phone` | 手机号 |
| `role_id` | 角色 |
| `status` | 启用/禁用 |
| `created_at` | 创建时间 |

#### `vehicles`

| 字段 | 说明 |
| --- | --- |
| `id` | 车辆 ID |
| `plate_no` | 车牌号 |
| `owner_id` | 车主 ID |
| `vehicle_type` | 车辆类型 |
| `color` | 颜色 |
| `created_at` | 创建时间 |

#### `intake_events`

| 字段 | 说明 |
| --- | --- |
| `id` | 接入事件 ID |
| `source_type` | `citizen/camera/admin` |
| `source_id` | 举报人、摄像头或管理员 ID |
| `location_text` | 地点描述 |
| `longitude` / `latitude` | 经纬度 |
| `captured_at` | 抓拍时间 |
| `speed` | 车速，可为空 |
| `image_hash` | 图片 hash |
| `status` | 接入状态 |
| `created_at` | 创建时间 |

#### `media_assets`

| 字段 | 说明 |
| --- | --- |
| `id` | 文件 ID |
| `intake_event_id` | 接入事件 |
| `asset_type` | `original/annotated/cropped_plate` |
| `url` | 文件访问地址 |
| `mime_type` | 文件类型 |
| `size` | 文件大小 |
| `created_at` | 创建时间 |

#### `cases`

| 字段 | 说明 |
| --- | --- |
| `id` | 案件 ID |
| `case_no` | 案件编号 |
| `intake_event_id` | 接入事件 |
| `status` | 案件状态 |
| `plate_no` | 车牌号，允许人工补录 |
| `violation_type` | 人工确认后的违章类型，审核前可为空 |
| `ai_review_id` | AI 初审结果 |
| `reviewer_id` | 人工审核人 |
| `review_opinion` | 人工审核意见 |
| `reviewed_at` | 审核时间 |
| `created_at` | 创建时间 |

#### `ai_detection_results`

| 字段 | 说明 |
| --- | --- |
| `id` | 检测结果 ID |
| `case_id` | 案件 ID |
| `model_name` | 模型名称 |
| `model_version` | 模型版本 |
| `detected_objects` | 检测到的目标 JSON，包括车辆、行人、信号灯、车道线等 |
| `object_confidences` | 各目标检测置信度 JSON |
| `plate_bbox` | 车牌框 JSON |
| `vehicle_bbox` | 车辆框 JSON |
| `raw_result` | 原始检测 JSON |
| `created_at` | 创建时间 |

#### `violation_rule_results`

| 字段 | 说明 |
| --- | --- |
| `id` | 规则判定结果 ID |
| `case_id` | 案件 ID |
| `candidate_violation_type` | 违章类型候选 |
| `rule_code` | 命中的违章规则编号 |
| `rule_matched` | 是否满足规则条件 |
| `evidence_level` | `complete/partial/insufficient` |
| `evidence_items` | 已具备证据 JSON |
| `missing_evidence` | 缺失证据 JSON |
| `reason` | 规则判定说明 |
| `created_at` | 创建时间 |

#### `ai_review_results`

| 字段 | 说明 |
| --- | --- |
| `id` | 初审结果 ID |
| `case_id` | 案件 ID |
| `review_mode` | `text_llm/vision_llm` |
| `conclusion` | `suggest_approve/need_review/suggest_reject` |
| `ai_confidence` | AI 初审置信度 |
| `reason` | 初审理由 |
| `risk_points` | 风险点 JSON |
| `missing_evidence` | 缺失证据 JSON |
| `prompt_version` | 提示词版本 |
| `created_at` | 创建时间 |

#### `violations`

| 字段 | 说明 |
| --- | --- |
| `id` | 正式违章 ID |
| `violation_no` | 违章编号 |
| `case_id` | 来源案件 |
| `vehicle_id` | 车辆 |
| `owner_id` | 车主 |
| `plate_no` | 车牌号 |
| `violation_type` | 违章类型 |
| `occurred_at` | 发生时间 |
| `location_text` | 发生地点 |
| `fine_amount` | 罚款金额 |
| `points` | 扣分 |
| `status` | `pending/handled/cancelled` |
| `created_at` | 创建时间 |

#### `notifications`

| 字段 | 说明 |
| --- | --- |
| `id` | 通知 ID |
| `violation_id` | 违章 ID |
| `owner_id` | 车主 |
| `channel` | `sms` |
| `phone` | 接收手机号 |
| `content` | 短信内容 |
| `status` | `pending/sent/failed/retrying` |
| `provider` | 短信服务商 |
| `provider_msg_id` | 服务商消息 ID |
| `retry_count` | 重试次数 |
| `sent_at` | 发送时间 |
