
## 八、Git 分支规范

```
main                         # 主分支，仅白沁禾合并
├── dev                      # 开发集成分支
│   ├── yangyi/xxx           # 杨翼 功能分支
│   ├── zhanghao/xxx         # 张浩 功能分支
│   ├── dingjingyi/xxx       # 丁璟怡 功能分支
│   └── tanggaopeng/xxx      # 唐高鹏 功能分支
└── release/v1.0             # 发布分支
```

- 每人从 `dev` 拉自己的功能分支开发
- 完成后提 PR 到 `dev`，由白沁禾或交叉 review 后合并
- 阶段里程碑通过后 `dev` → `main`
