# 🚦 智能交通违章管理平台

> **AI 驱动的交通违章智能管理平台**

本项目是一个现代化的 Web 交通违章管理平台，融合人工智能识别与智能数据管理能力。系统通过 AI 辅助检测、集中化管理、统计分析与可视化展示，提升交通违章处理效率，降低人工审核与管理成本。

---

## 📖 项目概述

智能交通违章管理平台是面向软件工程实践开发的智能交通管理系统，提供从违章证据上传、AI 辅助识别、车辆管理、驾驶员管理、违章记录管理、统计分析到系统管理的完整业务流程。

项目采用 **Vue 3**、**FastAPI** 与 **YOLOv8** 构建轻量级前后端分离应用，并具备 AI 图像识别辅助能力，适合作为智慧交通、违章管理与数据分析方向的课程实践项目。
